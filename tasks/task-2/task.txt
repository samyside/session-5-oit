Задание 2

	Построение кода сжатия информации
Описание задания
По результатам, полученным в предыдущей лабораторной работе, по строить коды сжатия информации используя метод Шеннона-Фэно (для тех у кого последняя цифра номера зачетки – четная) или метод Хаффмена (для тех у кого эта цифра – нечетная). Построение произвести, создав программу на языке Pascal или С.
Имя результирующего файла аналогично имени файла, в котором хранились частоты, но расширение .Haf если кодировка проводилась методом Хаффменна, или .S_F если кодировка проводилась методом Шеннона-Фэно. Например, обрабатывая файл doc01.tab методом Хаффмена получим файл с кодовой таблицей с именем doc01.Haf, если же этот файл обработан методом Шеннона-Фэно, то в результате получим файл с именем doc01.S_F.

	Содержание результирующего файла:

1. первая строка — имя исследуемого информационного файла;
2. вторая строка — количество байт в исследуемом файле;
3. пустая строка;
4. последующие строки — коды символов и коды сжатия, разделенные пробелом или символом табуляции;
5. пустая строка
6. средняя длинна кода сжатия
7. последняя строка — величина энтропии.

	Содержание отчета
1) Тема
2) Описание задания
3) Указать для каждого из исходных файлов значения энтропии Н, а также величину M = (H × N) / 8, где N — размер исходного файла. Какие
файлы могут иметь наибольшую степень сжатия (обосновать эти выводы). Какую информацию несет величина М ?
4) Текст программы на языке Pascal или С для построения кодов для сжатия информации с комментариями. Будет приветствоваться, если эти программы будут написаны студентами самостоятельно. В случае, если студенту трудно написать такую программу самостоятельно, то можно воспользоваться предлагаемыми вариантами программ.
5) Сделать выводы.
6) К отчету приложить таблицы кодировки символов для каждого из исходных файлов.

Например после обработки файла с частотно таблицей doc01.tab программой SHEN_FEN.PAS получим файл DOC1.S_F следующего содержания:

	Замечания
1. Тексты программ прилагаются. Файл SHEN_FEN.PAS содержит программу кодировки по методу Шеннона-Фэно, а файл HAFFMEN.PAS — Хаффмена. В тексте программ присутствуют незаполненные комментарии {1 }, {2 } и т.д. Студентам необходимо заполнить эти комментарии, т.е. указать, что именно выполняют команды, лежащие за этим комментарием, если он в пустой строке, или если он в строке с оператором — что выполняет этот оператор.
2. Предлагаемые программы написаны в среде PascalABC.Net. Если Вы используете другую среду, то, возможно, придется адаптировать текст программы под имеющуюся у Вас среду программирования.
